L.I.G.T.A.S. 
Location Intelligence & Geospatial Triage for Accelerated Support

Queue Tab:
Filter the citizens by selecting City, Barangay, and their Priority (Critical, High, Elevated, Stable) to specify the location and the target citizens.
Dispatch Route Button - Choose the asset to assign and dispatch to the queued citizen, the admin is able to reassign to other assets.
Locate Button - Immediately zoom in to the pinned citizen.
Mark Rescued Button - Update the status of the citizen.
Assets Tab:
Add Assets - Add assets to the map with their Asset Name, Type of Vehicle, Unit/Agency, Contact Number, Address, Latitude and Longitude by dragging and pin on the map.
Track Live Button - Zoom in to the location of the asset on the map.
Set Active Button - Update the status of Asset
Awaiting Dispatch - List of citizens that’s waiting for dispatch.
